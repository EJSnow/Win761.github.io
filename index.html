FOSS programs I support

Open-Shell Menu

Open-Shell is a free and open-source program for Windows that modifies the shell (Windows Explorer). The signature feature is Open-Shell Menu, the start menu replacement. It features Windows 7, XP, and 2000 start menu layouts.

WingetUI

WingetUI is a package manager GUI for Windows. It supports many package managers, including WinGet, the built-in CLI package manager on Windows. NOTE: Version 3.01 has some issues with lanuching, see here for details.

Notepad++

Notepad++ is a FOSS text editor, with the goal of being as lightweight and powerful as possible. It features advanced syntax highlighting, and extreme efficiency, even when working with large files.

Supermium

Supermium is a Chromium based browser with the goal of supporting as many versions of Windows as possible. Currently, Windows XP, Vista, 7, 8 and 8.1 are supported, and it obviously works on Windows 10 and 11. It's a little buggy though.

Windows 7 is the best operating system Microsoft ever made. I strongly support all efforts to keep it usable. Aero UI is the best thing ever.
